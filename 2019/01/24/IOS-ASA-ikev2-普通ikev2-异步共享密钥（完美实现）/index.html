#R2—-ASA之间建立ikev2 VPN

##下拓扑图在ASA于R2之间实现ikev2 VPN的同步密钥认证和异步密钥认证

R2:

IKEV2配置步骤：
一阶段：
1.配置ikev2 proposal建议

1.1 encryption
1.2 integrity
1.3 group14
1.4 prf
1-1.1调用proposal和匹配local接口地址 *

2.配置ikev2 keyring密匙串
2.1 peer
2.2 VPN对端IP，address
2.3 hostname
2.4 认证方式，PSK
3.配置ikev2 profile
3.1 匹配对端
3.2 本地标志
3.3 对端认证方式
3.4 本地认证方式
3.5 调用密匙串
二阶段：
4.配置IPSec transform转换集，配置感兴趣流量
5.配置IPSec Map
5.1 调用transoform
5.2 调用ikev2-profile
5.3 set对端peer
5.4 match 感兴趣流量
6.出接口调用MAP.

ASA：

IKEV2配置步骤：
一阶段：
1.配置ikev2 policy
1.1 encryption
1.2 integrity
1.3 group14
1.4 prf
2.配置ikev2 policy开启在接口
二阶段：
4.配置ipsec-proposal转换集
4.1 protocol esp encryption
4.2 protocol esp integrity
5.配置感兴趣流量 ACL
6.配置MAP
6.1 match address
6.2 set peer 对端
6.3 set ikev2 ipsec-proposal 调用proposal
6.4 interface 启用map
注意ASA还需要配置NAT，内部流量做免除
7.发现不通,做匹配的感情去流量做NAT免除
